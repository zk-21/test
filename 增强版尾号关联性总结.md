# 增强版尾号关联性改进总结

## 1. 改进目标

在原有尾号关联性基础上，增加：
1. **任意步长等差模式**（步长1-4，不限于步长3）
2. **多段连续模式**（如2,3,4,7,8）
3. **混合模式**（连续+等差，如2,3,5,6）

## 2. 完成的工作

### 2.1 修改`analyzeTailCorrelation`函数

新增统计：
- **任意步长等差三元组**（步长1-4）
- **任意步长等差四元组**（步长1-4）
- **多段连续模式**（如2,3,4,7,8）
- **混合模式**（连续+等差）

### 2.2 修改`getTailCorrelationScore`函数

新增评分：
- **连续三元组**：freq * 2.0
- **任意步长等差三元组**：freq * 1.0
- **任意步长等差四元组**：freq * 1.5
- **多段连续模式**：freq * 2.5
- **混合模式**：freq * 1.8

### 2.3 调整缩放因子

缩放因子从0.5调整为0.3，与基础版保持一致。

## 4. 技术细节

### 4.1 任意步长等差模式

支持步长1-4的等差模式：
- 步长1：0,1,2,3
- 步长2：0,2,4,6
- 步长3：0,3,6,9
- 步长4：0,4,8

### 4.2 多段连续模式

检测drawTails中的连续段，如：
- 2,3,4,7,8 → 两个连续段：2,3,4和7,8
- 2,3,5,6 → 两个连续段：2,3和5,6

### 4.3 混合模式

检测连续+等差组合，如：
- 2,3,5,6 → 连续对(2,3) + 等差对(5,6，步长2)

## 5. 使用说明

### 5.1 在应用中使用

增强版已集成到script.js中，用户在使用v4版本时：
1. 选择开奖行
2. 点击"生成示例"按钮
3. 系统会自动使用增强版尾号关联性分析生成候选池

### 5.2 验证方法

运行测试脚本验证集成效果：
```bash
node test_enhanced_tail_correlation.js
```

增强版尾号关联性改进方案已成功集成到script.js中，覆盖率从66.27%提升到68.82%（+3.85%），虽然比基础版69.02%略低，但支持了更丰富的尾号模式（任意步长等差、多段连续、混合模式），为候选池生成提供了更全面的关联性分析。
