# 补漏6候选池覆盖率优化报告

## 问题诊断

### 原始问题
- 补漏6候选池是整个25球池（`result.pool`）
- 高频号优先策略导致补漏6选择Top5已覆盖的号码
- 联合覆盖率完全没有提升（+0球）

### 根本原因
```javascript
// 旧代码：候选池是整个25球池
const candidate6Scored = result.pool
  .map(e => { ... })  // 包括Top5已覆盖的号码
```

## 解决方案

### 修改策略
```javascript
// 新代码：只选Top5未覆盖的号码
const candidate6Scored = result.pool
  .filter(e => !top5CoveredNums.has(e.number))  // 补盲区
  .map(e => { ... })
```

### 补充逻辑优化
当未覆盖号码不足5个时：
1. 先补未覆盖池号码
2. 再补已覆盖池号码（打5折权重）

## 回测结果对比

| 指标 | 旧策略（高频号优先） | 新策略（补盲区优先） | 变化 |
|------|---------------------|---------------------|------|
| 补漏6命中率 | 154/495 (31.1%) | 103/495 (20.8%) | -10.3pp |
| Top5+补漏6联合覆盖 | 333/495 (67.3%) | 436/495 (88.1%) | **+20.8pp** |
| 补漏6增补覆盖 | +0球 (0.0%) | +103球 (20.8%) | **+103球** |
| 补漏6命中分布 | 5球1,4球2,3球11,2球31,1球46,0球8 | 5球0,4球0,3球4,2球23,1球45,0球27 | - |

## 关键发现

1. **补漏6的真正价值**：不是独立命中率，而是补充Top5的盲区
2. **高频号策略的陷阱**：选择Top5已覆盖的号码，联合覆盖率零提升
3. **补盲区策略的优势**：虽然单注命中率下降，但整体覆盖率大幅提升

## 策略建议

- ✅ **推荐使用补盲区策略**：联合覆盖率88.1% vs 67.3%
- ⚠️ **高频号策略**：仅适用于追求单注命中率的场景
- 💡 **混合策略**：可以考虑70%补盲区 + 30%高频号的组合
